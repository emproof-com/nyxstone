Detailed Description

Records the ordered sequence of emitLabel / emitInstruction callbacks the assembly parser issues, deferring all encoding to a single post-parse pass run by Nyxstone::assemble_impl. By skipping the full MC object-file emission pipeline, the assembler avoids the per-call ELF section/symbol bookkeeping that dominated the previous path's profile.
