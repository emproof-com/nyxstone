Detailed Description

Minimal MCObjectFileInfo replacement: only registers .text with the MCContext, skipping the ~40 other section creations performed by MCObjectFileInfo::initELFMCObjectFileInfo and the LLVM createMCObjectFileInfo factory. Nyxstone restricts its output to .text, so the parser never queries other sections in practice.
